=LEXVAN version alpha, oct. 2013=
_________________________________

==Contenu==

  * Vue_Candidats_VN_x_validated_30-10-2013.csv: extraction de la base LexVAN, contenant les associations V->N validées.
  Une paire est validée à partir du moment où, pour un V donné, un N morphologiquement apparenté et attesté dans Delaf, Wiktionary, Verbaction ou autre lexique de référence a été trouvé. Ce fichier ne contient que des paires validées par Antonio Balvet.
  * wiktionaryXfr2010_4Lexvan_formatted-ID_25-06-2013_final.csv: conversion en texte tabulé du dump XML du wiktionary, préparé par CLLE-ERSS (WiktionaryXfr2010) pour exploitation dans le lexique LexVAN. Les informations essentielles conservées sont: la catégorie grammaticale, l'id dans wiktionary, le lemme, le nombre de sens, les liens sémantiques (synonymie, antonymie, hyper-/hyponymie, méronymie/holonymie) et les traductions. Le fichier contient 160309 entrées, réparties en:
    * N		102659
    * V		17875
    * ADJ	39776.
